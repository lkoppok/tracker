v1.0 - 核心功能搭建
基础架构：建立基于 HTML/CSS/JS 的单页 Tracker，数据本地化存储 (LocalStorage)。

双重追踪：支持“书籍”与“网课”两种模式，记录总进度与当前进度。

视觉风格：确立“纸墨风”极简设计，支持系统级暗色模式 (Dark Mode)。

v2.0 - 云端同步与数据结构优化
Google Sheets 联动：通过 Google Apps Script 实现后端同步。当项目完成度达到 100% 时，自动将结果存入云端表格。

字段增强：增加“作者”字段录入，为后续数据分析提供维度。

逻辑修正：修复了暗色模式下输入框文字看不清的 UI Bug。

v3.0 - 时间维度与打卡系统 (Current)
🔥 Streak 打卡算法：引入连续打卡逻辑，右上角实时显示连续天数，点击可查看累计打卡总天数。

📅 动态日历：新增迷你日历视图，自动标注阅读日期（橘色圆点），可视化阅读频率。

🧟 僵尸项目自动清理：

增加“正在进行中”动态计数，如 正在进行中 (3)。

自动沉淀逻辑：超过 15 天未更新进度的项目会自动变灰并移动至“停滞区”，保持主界面专注。

UI 细节优化：

Streak 区域背景透明化，完美适配深夜阅读环境。

增加“唤醒”机制，一键激活停滞项目。
